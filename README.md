This repository contains exploratory data analysis using publicly available datasets.

Topics analyzed include:

* Fiscal conditions in large economies
* Global energy consumption
* Tree cover loss in Peru

The analysis is conducted using Python and Jupyter notebooks (Google Colab).

## Repository structure

notebooks/
01_fiscal_crisis_analysis.ipynb
02_global_energy_consumption.ipynb
03_peru_tree_cover_loss.ipynb

## Data sources

Public datasets including:

* Our World in Data
* International Monetary Fund (IMF)
* Global Forest Watch
